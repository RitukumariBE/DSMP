# Python 
This repository contains my Python learning journey,starting from installation to fundamental programming concepts.
It includes nots,examples, and practice programs.

## 1. Installing Python
### Step 1.Download Python
1. Go to the official Python website.<br>
   https://code.visualstudio.com/
2. Click Download Python

### Step 2.Run the Installer
- Open the downloaded file
- IMPORTANT: Check the box:
  ```
  Add Python to PATH
  ```
Then click Install Now

### Step 3. Verify Installation
Open Command Prompt/Terminal and type:
```python
python --version
```
or
```
python3 --version
```
If Python is installed correctly,you will see something like:
```
3.12.2
```
## 2. Installing VS Code(Recommended)

## 4. Running your First Python Program
Create a file :
```python
hello.py
```
Add this code:
```python
print("Hello World'")
```
Run the program using terminal:
```python
python hello.py
```
Output:
```pyhon
Hello World
```
